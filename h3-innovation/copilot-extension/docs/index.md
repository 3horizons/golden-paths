# GitHub Copilot Extension

**Horizon:** H3 — Innovation  
**Tags:** GitHub Copilot, Copilot Extension, AI Agent, GitHub App

## Overview

GitHub Copilot Extension (Agent or Skillset) with custom tools, Azure data source integration, and OAuth auth.

## What You Get

- GitHub App configuration (`app.yml`) with Copilot Extension manifest
- Agent or Skillset implementation (Node.js / Python)
- OAuth 2.0 authentication flow (GitHub user auth)
- Azure integration: Key Vault, Cognitive Search, or REST API tool
- Skill definitions (`skills.json`) with schemas
- Express.js server handling `/_chat` and `/_discovery` endpoints
- Deployment to Azure App Service or AKS
- GitHub Actions: CI build + App Service deploy workflow
- Catalog registration
