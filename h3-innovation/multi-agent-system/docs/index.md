# Multi-Agent System

**Horizon:** H3 — Innovation  
**Tags:** Multi-Agent, AutoGen, Semantic Kernel, Orchestration, AI

## Overview

Production multi-agent AI system with orchestration, agent collaboration, human-in-the-loop, and conversation management.

## What You Get

- Orchestrator agent definition with sub-agent routing
- Specialist agent templates (researcher, coder, reviewer, planner)
- Human-in-the-loop integration with approval gates
- Conversation history and state management
- AutoGen or Semantic Kernel framework setup (user's choice)
- Azure AI Foundry deployment for all agents
- Web UI for human oversight (Chainlit or Gradio)
- GitHub Actions: deploy all agents on merge
- Catalog registration
